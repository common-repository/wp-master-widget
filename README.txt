WP Master Widget is an advanced WordPress widget that allows easy styling and organization for text, fontawesome icon, image, and more types of elements.

== Description ==

Supports vertical ordering of images, icons, texts, titles, and more using a simple drag and drop ordering feature. Users can dynamically add any number of elements to the WP Master Widget and each element can be styled differently from the "Widgets" page. All widgets can be configured as a hyperlink to any destination with styling options.

* Image widget allows using hover animation based on pure CSS.
* Text Widget allows using shortcodes and simple styling available from the user interface.
* Icon Widget loads FontAwesome icons library with style settings.
* All Widgets can be configured with hyperlink with options to open from a new window and styling.

== Installation ==

1. Upload 'wp-master-widget' folder to the '/wp-content/plugins/' directory or select 'wp-master-widget.zip' to upload via WordPress Plugin Installer.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "AAppearance > "Wdigets" > select "WP Master Widget."
